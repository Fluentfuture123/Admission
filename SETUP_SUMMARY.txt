🔐 DATA STORAGE SYSTEM

How It Works:
• Form data is stored in browser's localStorage
• No data is sent anywhere unless Google Sheets is configured
• Each submission gets unique admission number (FF-1001, FF-1002...)
• Data persists in the browser until you clear it

Backup Your Data:
1. Open admin.html
2. Click "Export to CSV"
3. Save the file to your computer
4. Can open in Excel or Google Sheets

Caution:
⚠️ Clearing browser data/cache will delete all stored submissions
📌 Backup regularly using the CSV export feature

⚙️ ADVANCED: SETTING UP GOOGLE SHEETS (OPTIONAL)

If you want Google Sheets integration instead of local storage:

Step 1: Create Google Sheet
- Go to Google Sheets
- Create new spreadsheet
- Name: "Fluent Future Admissions"
- Add these column headers:
  Admission Number | Name | Age | Gender | Email | Phone | School | Subjects | Grade | Comments | Timestamp

Step 2: Create Google Apps Script
- In your Google Sheet, go to Extensions → Apps Script
- Replace all code with the code in README.md
- Deploy as Web App
- Copy the URL it gives you

Step 3: Update Your Website
- Open script.js
- Find: var WEB_APP_URL = 'https://script.google.com/macros/s/...'
- Replace with your Google Apps Script URL
- Save file

After that, submissions will go to both Google Sheets and local storage!

❓ TROUBLESHOOTING

Q: Images are not showing?
A: 1. Check IMG folder exists
   2. Verify filenames are exact match (case-sensitive)
   3. Try opening HTML file directly in browser

Q: Form won't submit?
A: 1. Fill all required fields (Name, Email, Phone, Age, Gender)
   2. Check browser console (F12) for errors
   3. Check admin.html to see if data saved locally

Q: Can't find admin.html?
A: 1. Open the website folder
   2. Look for admin.html file
   3. Double-click to open it

Q: How do I backup my data?
A: 1. Open admin.html
   2. Click "Export to CSV"
   3. Choose where to save file
