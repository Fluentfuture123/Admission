📄 DOWNLOADING AS PDF (A4 SIZE)

Why PDF?
✓ Professional format like a CV
✓ Easy to print
✓ Perfect for record keeping
✓ Can be emailed to others
✓ A4 paper size (standard)

How to Download:
1. Open admin.html
2. Click "View Details" on a submission
3. Click "📄 Download PDF (A4)" button
4. File downloads automatically as:
   admission-[ADMISSION-ID].pdf
   Example: admission-FF-1001.pdf

Where does it go?
→ Usually goes to your Downloads folder
→ Or wherever you set as download location

Tips:
• PDF is formatted nicely like a CV
• Has sections for each type of info
• Can print directly to paper
• Best for official records

🔒 DATA STORAGE & SECURITY

Where is my data stored?
→ Locally in your browser's localStorage
→ On your computer only
→ NOT sent to any server by default

Is it safe?
✓ YES - Data only on your computer
✓ Data won't be lost if internet goes down
✓ Only you have access to the data

When should I backup?
✓ Monthly backup recommended
✓ After receiving many submissions
✓ Before clearing browser data
✓ Before deleting the website folder

How to backup?
1. Open admin.html
2. Click "📥 Export All to CSV"
3. Save the file to a safe location
4. Or save to Google Drive/OneDrive

⚠️ IMPORTANT WARNING:
→ If you clear browser cache/data, you'll lose all submissions
→ Always export/backup before clearing browser data
→ Keep multiple backup copies

❓ FREQUENTLY ASKED QUESTIONS

Q: Can I edit a submission after it's submitted?
A: Currently no. You can only view. To edit, manually update the CSV file.

Q: What if I accidentally cleared the data?
A: If you have a CSV backup, you can re-import it. Otherwise, data is lost.

Q: Can students see the admin page?
A: No, it's a separate file. Don't share the admin.html link with students.

Q: How many submissions can I store?
A: Theoretically unlimited, but browser localStorage typically stores up to 5-10MB of data (thousands of submissions).

Q: What file format is the PDF?
A: Standard PDF format. Can open with any PDF reader (Adobe, Preview, etc.)

Q: Can I print the PDF directly?
A: Yes! PDF is formatted for printing. Click Print while viewing PDF.

Q: What if the PDF/Image download doesn't work?
A: Check browser console (F12) for errors. Try a different browser.

Q: How do I organize downloaded files?
A: Create folders by month/year:
   → 2026-06-Admissions/
   → 2026-07-Admissions/
   → etc.
